What is Telegraph?

Telegraph is Telegram based client. All of your chats are synced between Telegram server and other Telegram clients. Telegraph is not including all Telegram features, it just has the main ones, But has some practical automated and statistic features that can help you to enjoy chatting. Telegraph is a layer between Telegram and you, so you can use it to break filtering in your country.

Features

Secure

Telegraph is using Telegram API, so it is as secure as Telegram. Telegraph is not storing any of your data, you can be sure that your data is safe.

Open Source

Telegraph is open source, so you can check the code and be sure that it is safe. It is available on GitHub. You can also contribute to the project.

Free

Because of our server costs, we are not able to provide free service for everyone. But we are trying to provide free service for some special users. For further information, please contact us.
